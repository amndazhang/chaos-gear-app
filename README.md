## CHAOS Gear App
Working repo for [CHAOS]([url](https://www.chaosberkeley.org/#/))' gear shed. Redesigning and redeveloping the club's gear exchange experience to be simple and accessible for all those who go outside. 

Member-facing app:
- Simple member info intake form --> Supabase member table
- Outputs a member uiud qr code

Officer-facing app
1. Gear checkout
- Scan member qr code
- Scan physical items to-borrow
3. Inventory item lookup
4. Member lookup
